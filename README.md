# Igumnov_minigame_zombie_shooter
#Шутер зомби, 
##Задачи:
1. убить 5 зомби,
2. за каждою победу плюс педесят кубков,
3. у зомби 60 хр,
4. зомби бьют по 10 хр, 
5. у нас  100 хр,
6. мы бьём по 20 хр.


## Ресурсы
1.контроллер
2.макетка
3.аккумулятор 
4. заряд аккума
5. 3 кнопки
6.  матрица 8 на 8
7. числовой индикатор  tm1637


## Состояния

1. ждать 
2. играть
3. победа
4. поражение
5. разрядился


## Способности
## игрок:
1. вверх 
2. вниз,
3. выстрел


## Враг
1. появиться
2. идти влево
3. кусать
 
## Устройство
1. проверить нажатие кнопки 1
2. проверить нажатие кнопки 2
3. проверить нажатие кнопки 3
4. зажечь пиксель матрицы
5. очистить матрицу
6. показать число на индикаторе
